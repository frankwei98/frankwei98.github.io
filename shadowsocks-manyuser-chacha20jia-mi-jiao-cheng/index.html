Shadowsocks & Chacha20加密教程

由于经常需要用到CHACHA20 故作本文

本文仅在CentOS 7下测试可行。其他系统请勿照搬

安装Shadowsocks-Manyuser

首先你得有个VPS！(废话)

cd /root
yum install python-setuptools && easy_install pip
yum install m2crypto git
pip install cymysql
git clone -b manyuser https://github.com/breakwa11/shadowsocks.git

然后修改Config.py里的数据库配置

下载Libsodium

wget https://download.libsodium.org/libsodium/releases/LATEST.tar.gz
tar zxf LATEST.tar.gz
cd libsodium* 
./configure
make
make install

如果执行 ./configure 出错

执行

yum install gcc

然后再重新执行

如果make install后没有提示错误

至此Libsodium算是安装完成了
但是我们还得修复动态链接库

vi /etc/ld.so.conf

在后面添加

/usr/local/lib

保存退出后，运行命令：

ldconfig

修改配置文件 使加密方式生效

vi /root/shadowsocks/config.json

把AES-256-CFB改为chacha20即可
