简化FinalSpeed在Mac OS X下的打开步骤

情况

FinalSpeed的客户端有Windows版和Java版，Java版支持OS X,Linux.

在Mac里FinalSpeed必须以root权限运行,因为没有root权限,会无法启用TCP协议.

而每次在终端操作太麻烦 所以想到了这个方法

那就是——写脚本!

方法

新建一个.sh文件，我设置的是finalspeed.sh

然后在文本编辑器里输入

#!/bin/sh
cd /Users/frank/Downloads/finalspeed_client/
sudo java -jar finalspeed_client.jar


保存后，打开Mac的终端

切换到finalspeed.sh所在的目录 (我放在桌面）

输入以下命令

chmod +x finalspeed.sh

以后只需要双击finalspeed.sh,输入密码验证即可
