一加手机3评测:也就值2499的「旗舰机」

  • android
  • oneplus
  • 一加
  • 手机
  • 评测

文章开始之前先给大家道个歉

拖来拖去真是不好意思呢

评测内容不全真是不好意思呢

高考过后准备入手一台android旗舰

恰逢一加手机发布了新品 于是我就买了

这次一加手机3和一加手机 X 一样采用了 AMOLED 屏幕材质，5.5寸1080p屏幕，效果只能算一般(P排的AMOLED PPI还要乘以0.82)，同时这个屏幕尺寸对于一些不喜欢“巨屏”的人来说是一个灾难，7.3mm 的厚度和金属后盖让手感更上一层楼。我拿出了我那部吃灰的MX 5 比较了一下，整机大小和MX5无异。

这一代和上一代一加手机2一样，采用了USB Type-C接口，不但在输入输出功率上比上一代提高并且支持正反插，这才是USB的未来啊！

然而很遗憾，一加手机3的USB Type-C并不是USB 3.0的标准 还是上一代一样采用USB 2.0

亮点

  • 支持支付宝和微信指纹支付 (国产Android手机 华为都支持 小米目前都不支持 魅族仅支持支付宝）
  • 支持NFC 意味着你可以当自己的手机是银行卡 、公交卡
  • 6GB RAM 一加三是目前使用高通820的手机中 内存最大的手机
  • 全网通+双卡双待 : 一加第一次推出了全网通机型

总结&评价

优点

  • 旗舰级配置 : 高通820+6GB RAM+64G ROM
  • 内核开源 : 官方开源内核 允许用户刷机
  • 全网通+双卡双待 : 电信用户的福音!!!终于可以一边用电信卡打电话 一边用联通卡上网了 (广东联通非常良心)
  • 非常容易抢 : 京东第一次抢购我花了3分钟就抢到了 闲鱼还有一堆人以原价出掉自己在京东抢购到的一加手机3
  • 指纹识别快 : 一摸指纹识别sensor就解锁了 识别速度和 iPhone 6S 有的一比
  • 外媒口中的「旗舰杀手」 : 瘾科技 、The Verge 和 Android Authority 都给予一加手机3不错的评价

缺点

  • 电量造假 : 一加3的快充电量造假 (因此我们放弃了充电对比环节)
  • 老板爱吹牛逼 : 一加在摄像上的调教并没有发布会上刘作虎吹的那么牛逼 (喜欢拍照的用户加几百块钱买三星 Galaxy S7 真心话)
  • 太「氢」了 氢OS简陋程度和原生Android有的一比 说得难听点就是个好看的花瓶
  • 不兼容的快充 : 与众多采用了高通820手机不同，一加 3 采用了 OPPO 的VOOC的快充方案而不是广泛采用的高通Quick Charge，意味着市面上大多数的快充头&充电宝都不兼容一加3的所谓「DASH极速闪充技术」

总结

我懒得码字了 直接引用 瘾科技的评价 吧

总的来说，一加 3 在软、硬件层面都能算是一台称职的当代旗舰，而且在手感和软件部分更是直中小编所求。所以如果说是「2016 年的旗舰杀手」，一加 3 在这短短的动手玩中已经能给小编这样的感觉了，只是问鼎年度机皇，还是多需要一点时间来观察。但考虑到官方依旧采用低定价的策略来销售一加 3，如此性价比，已经大大增加了它对一般消费者的吸引力了。
